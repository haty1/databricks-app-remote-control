# Databricks App Controller

This is a simple HTML/JavaScript app to allow business users to start, stop, and check the status of a Databricks app using the Databricks Apps REST API.

## How to use

1. Open the index.html in a web browser.
2. Enter your Databricks workspace URL, personal access token, and the app ID (app name).
3. Use the buttons to start, stop, or get the status of your app.
4. To embed in Power BI, use an HTML content custom visual and point it to this interface.

## Files

- index.html: The main user interface.
- app.js: JavaScript logic for API calls.
- style.css: Basic styling for the interface.
- README.txt: This instructions file.

## Notes
Be sure to keep your personal access token secure.
Use HTTPS when possible.
